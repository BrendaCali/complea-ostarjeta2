# Carpeta de Música

Sube aquí tu archivo de música de 51 segundos.

## Instrucciones:

1. El archivo debe llamarse exactamente: `cumpleanos.mp3` (o el formato que prefieras: .mp3, .wav, .ogg)
2. Súbelo a esta carpeta `/public/music/`
3. El código ya está configurado para reproducirlo automáticamente

**Nota:** El código actual usa `cumpleanos.mp3` como nombre de archivo. Si usas otro nombre, actualiza el código en `/src/app/App.tsx`.
